PROJECTS

During my time at S G Balekundri Institute of Technology, I had the opportunity to work on a remarkable projects

Project 1:

Vehicle Pollution Monitoring System using IoT

This project consisted a detection device which could be mounted on vehicle itself to minimize the pollution in metropolitan areas, as it is crucial to track vehicle emissions and monitor them. The main aim of this project was to create a valuable route for sensing engine emissions and reduce the emissions in the atmosphere.The kit contained various hardware components such as PIC controller, pollution sensor, relay, relay drivers, buzzer and wi-fi modem.The main program was embedded in PIC controller.
